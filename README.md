# **Remote Desktop Viewer**

A Python-based application for real-time remote desktop sharing, chat, and file transfer over a network.

---

## **Features**

- **Real-Time Screen Sharing**: View a remote desktop in real-time.(done)
- **Chat Functionality**: Send and receive messages seamlessly.
- **File Transfer**: Easily transfer files between the server and client.(done)

---

## **Technologies Used**

- **Python**: Core programming language.
- **PyQt6**: For building the client-side GUI.
- **Pillow (PIL)**: For image processing during screen sharing.
- **Socket Programming**: For server-client communication.

---

## **Installation**

1. Clone this repository:
   ```bash
   git clone <repository-url>
   cd <repository-folder>
   ```

2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Ensure Python 3.9 or higher is installed on your system.

---

## **Usage**

### **Start the Main**

1. You get the option to whether give access to become remote or take access of another pc:

### **REMOTE**
1. Your IP Address will be displayed.
2. Enter an open port on your pc
3. Click on Open to Connect
4. The server will start listening on `Your_IP`.

### **TAKE ACCESS**
1. Enter the IP and Port in their respective fields.
2. Click on Connect.
3. Help Out your friend now!!!
